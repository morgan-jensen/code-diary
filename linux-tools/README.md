# Linux Tools That I Like

* [fd](https://github.com/sharkdp/fd) is a nicer version of the `find` command.
    * Downloaded originally as a dependency for nvim/treesitter, but it works quite nice from the cmd line.
* [ripgrep](https://github.com/BurntSushi/ripgrep) is a nicer version of `grep` command.
    * Also originally a dependency for nvim/treesitter.
    * Formats results very nicely and very readable.
