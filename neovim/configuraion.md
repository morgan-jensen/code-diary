All information is accurate as of neovim version 0.12.2.

## Modules and Features

### LSP
Information found from [neovim.io](https://neovim.io/doc/user/lsp/#lsp-quickstart).

Nvim provides an LSP client, but the servers are provided by third parties. Follow these steps to get LSP features:

1. Install language servers using your package manager or by following the upstream installation instructions. Language servers can be found [here](https://microsoft.github.io/language-server-protocol/implementors/servers/).
2. Define a new config lsp-new-config (or install [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)). See neovim documentation linked at top of section for example.
